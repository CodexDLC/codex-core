# 🛠️ Common Utilities API

Технический справочник общих утилит библиотеки. Выберите подраздел в меню слева для просмотра деталей.

- **Logger**: Стандартизированное логирование.
- **Phone**: Парсинг и валидация телефонных номеров.
- **Cache**: Утилиты для кэширования.
- **Text**: Обработка текста и строк.
