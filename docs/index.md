# Codex Core Documentation

[🛡️ Architecture (EN)](en/architecture/README.md) | [⚙️ API Reference](en/api/index.md) | [🗺 Roadmap](evolution/roadmap.md)

Welcome to the documentation for **codex-core**, a foundational library used across all Codex ecosystem projects. It provides:

- **🛡️ Core PII Masking & DTOs**: Standardized immutable data models with automated sensitive data protection.
- **🛠️ Common Utilities**: Reliable normalization for phone numbers, names, and structured logging.
- **⚙️ Settings Architecture**: Base configuration patterns for standardized infrastructure management.
